### Verschiedene Arten einen String zu deklarieren
- Klassisch
    - Sonderzeichen escapen mit '\\'

    - "Te\\\\st"
    - "er Sagte \\"Hallo\\""
- Backtick -> Javascript Template
    -  \`Test\`
    - \`Multi 
      line\`
    - \`x ist : ${x}\`
- String-Function
    - String(42) -> "42"
- Regex
    - /hallo/
